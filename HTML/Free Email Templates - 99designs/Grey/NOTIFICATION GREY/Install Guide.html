Adding a Device to MarWatch

 


Step 1

When creating a new device it will be placed in the container level you are currently in. For the example we are in the root container any new device or container would be placed here. If you'd like to create a device for a particular container navigate to it before creating a new device.





Step 2

Once you've navigated to your desired location hover over the plus icon in the top left and select 'New Device', highlighted yellow in the example.









Step 3

For each session you will have to authenticate your administrate privileges, the password will be the same as your login.











Step 4
From the drop down you can select the device type you wish to create for this example we are going to create a probe.











Step 5
Configure your device for this example we used a probe as it will need to be created first. For a probe all you will need is a valid name. Best practise would be something descriptive like "Customer A DataCenter". Information needed for set-up includes your devices IP Address, SNMP, MiXML and any associated Username and Passwords.





Step 6
Once created navigate to the probe's page and you'll have the probe software gadget, highlighted in red. It'll have all the versions of software you'll need. When your installing the probe software the url need for configuration is highlighted in orange, you will still need this url if you are using a hardware probe.

If you made a mistake configuring your device or have licensed new features they can be changed at any time. Navigate to the gear icon and select setting to modify any of your devices information or select licenses to attach a license to the device.
